# Agent-Aiko

## ディレクトリ構成

```
agent-aiko/
├── docs/
│   ├── manga-document.md           # 漫画全ページの構造化ドキュメント
│   ├── aico-p0-persona-spec.md     # アンドロイドとしての人格仕様書（実装用）
│   ├── aiko-persona-protocol.md    # 人格プロトコル初版（参照用）
│   └── CLAUDE-v1-draft.md          # エージェント定義 v1（作業中）
├── logs/
│   └── dev-log.jsonl               # 開発ログ
└── images/                         # 原典漫画データ（Git管理外）
```
